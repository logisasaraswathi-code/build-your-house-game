# build-your-house-game
A probability-based mini-game where players invest virtual money to build a house.
# Investment Round - Build Your Dream House

## Game Concept
Start with ₹1,000 and invest money in probability-based market rounds. Earn money to build a house! Goal: ₹30,000 to complete the house.

## How to Play
1. Choose an investment amount: Small, Medium, or All-in
2. Market will randomly go Up or Down
3. Gain or lose money based on probability
4. Accumulate ₹30,000 to build your dream house

## Tech Stack
- HTML, CSS, JavaScript
- No backend required (self-contained)
